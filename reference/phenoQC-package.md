# phenoQC: Automated Phenotypic Data Quality Control for Plant Breeding Trials

Provides automated, reproducible quality control for phenotypic data
from plant breeding field trials. Includes trial structure validation
(duplicate plots, missing entries, replication balance), statistical and
spatial outlier detection, spatial trend diagnostics, missing data
analysis with spatial imputation, and automated HTML QC report
generation. All functions are pipe-friendly and non-interactive,
designed for scripting and batch processing. Integrates seamlessly with
'brapiR2' for end-to-end breeding data pipelines.
